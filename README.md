# Sistema de Control de Inventario en Python

## Descripción
Este proyecto corresponde a la Fase 5 – Evaluación Final POA del curso Fundamentos de Programación de la Universidad Nacional Abierta y a Distancia (UNAD).

El programa fue desarrollado en Python y permite analizar el inventario de artículos, identificando cuáles productos requieren reabastecimiento según el stock mínimo establecido.

## Funcionalidades
- Registro de artículos mediante matrices.
- Uso de funciones en Python.
- Validación de stock mínimo.
- Cálculo automático de reabastecimiento.
- Generación de reporte de inventario.
